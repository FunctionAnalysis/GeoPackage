GeoPackage Libraries

by the National Geospatial-Intelligence Agency

The National Geospatial-Intelligence Agency (NGA), in collaboration with BIT Systems, makes available under the MIT License a collection of OGC GeoPackage spec libraries listed as GeoPackage implementations. The NGA GeoPackage Libraries include creation, visualization, and management SDKs including sample applications for Android and iOS.

GeoPackage Java

A Java library that provides the ability to read, create, and edit GeoPackage files.

        

GeoPackage Android (iOS coming soon)

A SDK that provides the ability to manage GeoPackage files providing read, write, import, export, share, and open support. Open GeoPackage files provide read and write access to features and tiles.
